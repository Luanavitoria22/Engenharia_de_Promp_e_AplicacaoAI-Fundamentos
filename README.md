# Engenharia_de_Promp_e_AplicacaoAI -  Repositório de Atividades

Bem-vindo(a) ao repositório da disciplina Engenharia de Prompt e Aplicações em IA!
Este espaço foi criado para apresentar e documentar todas as atividades, estudos e experimentações desenvolvidas ao longo da matéria.

🎯 Objetivo do Repositório

O objetivo deste repositório é reunir exercícios, práticas guiadas, desafios e projetos realizados durante o curso, demonstrando minha evolução no desenvolvimento de prompts eficientes, estruturados e alinhados às boas práticas de interação com modelos de linguagem.
