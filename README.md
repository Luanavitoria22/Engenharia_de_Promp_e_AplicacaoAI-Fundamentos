# Engenharia_de_Promp_e_AplicacaoAI -  Repositório de Atividades

Bem-vindo(a) ao repositório da disciplina Engenharia de Prompt e Aplicações em IA!
Este espaço foi criado para apresentar e documentar todas as atividades, estudos e experimentações desenvolvidas ao longo desse semestre.

🎯 Objetivo do Repositório:

O objetivo deste repositório é reunir exercícios, práticas guiadas, desafios e projetos realizados durante o curso, demonstrando minha evolução no desenvolvimento de prompts eficientes, estruturados e alinhados às boas práticas de interação com modelos de linguagem.

📚 Estrutura da Disciplina:

A disciplina está organizada em quatro unidades, que abordam desde fundamentos até aplicações avançadas de IA:

- 🧩 **Unidade I — De Conversa Passiva a Diálogo Inteligente:** Tornando-se Arquiteto da IA
- 💻 **Unidade II — IA como Parceira de Código:** Programação Assistida e Automação
- 🧱 **Unidade III — Low Code, No Code e Vibecode:** Do Código Tradicional à Democratização de Programação com IA
- 🤖 **Unidade IV — Criação de Agentes de IA e Ética Aplicada:** Da Inovação Responsável ao Impacto Social – como Construir Inteligências que Refletem nossos Valores

📌 Considerações:

Este repositório funciona como um registro da trajetória de aprendizado, servindo tanto como portfólio quanto como material de consulta para futuros projetos envolvendo Inteligência Artificial.
